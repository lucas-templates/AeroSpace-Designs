# AeroSpace Designs - Single Page Landing

A modern, responsive single-page landing website showcasing innovative micro-apartment designs with interactive 3D tours and a modular furniture configurator.

## üéØ Project Overview

AeroSpace Designs is a single-page landing website that demonstrates the concept of "Big Living in Smart Spaces" - showcasing how small urban spaces can be optimized for maximum comfort and functionality. The website features immersive 3D apartment tours, an interactive furniture configurator, and a modern architectural design aesthetic.

## ‚ú® Features

- **3D Apartment Tours**: Interactive 3D visualizations of three different micro-apartment layouts:
  - Studio 250 (250 sq ft)
  - One Bedroom 350 (350 sq ft)
  - Loft 400 (400 sq ft)
- **Modular Furniture Configurator**: Drag-and-drop interface for designing custom space layouts with modular furniture pieces:
  - Murphy Bed
  - Wall Desk
  - Storage Units
  - Kitchen Modules
  - Convertible Sofa
- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Smooth Scrolling Navigation**: Elegant smooth scrolling between page sections
- **Interactive Contact Form**: Contact form for user inquiries
- **Modern UI/UX**: Clean, architectural-themed design with smooth animations and transitions
- **3D View Controls**: Rotate and zoom functionality for apartment tours

## üöÄ Setup Instructions

### Prerequisites

No build tools or dependencies are required. The project uses CDN links for Bootstrap and Bootstrap Icons, so you only need a modern web browser.

### Installation

1. **Clone or download the project** to your local machine

2. **Open the project folder** in your file explorer

3. **Open `index.html`** in a web browser:
   - Double-click the `index.html` file, or
   - Right-click and select "Open with" ‚Üí your preferred browser, or
   - Use a local development server (optional but recommended)

### Optional: Using a Local Development Server

For the best experience, you can use a local development server:

**Using Python:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**Using Node.js (with http-server):**
```bash
npx http-server
```

**Using PHP:**
```bash
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.
